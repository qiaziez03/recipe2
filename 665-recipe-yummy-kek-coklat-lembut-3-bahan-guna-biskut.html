Kek Coklat Lembut 3 Bahan Guna Biskut instructions

  1. Asingkan biskut dan inti..
  2. Letak biskut ke dalam blender tuang 1/4 cawan susu full cream, lebih sikit pun boleh. Kisar hingga halus, tuang ke dalam mangkuk yang selesa. Kemudian letak baking powder kacau sebati..
  3. Ambil intinya masukkan ke dalam blender, tuang sebahagian susu full cream tadi dan kisar hingga halus. Tuang sedikit-sedikit ke dalam adunan pertama tadi, sebatikan dan ratakan. Buat hingga habis..
  4. Sediakan saucepan saiz 7 inci, sapu minyak lapik baking paper. Tuang adunan letak atas dapur, tutup dengan penutup periuk bakar 30- 40 minit api perlahan. Jika bakar guna api dapur, letakkan satu lagi stove stand jadi double supaya api tak terlalu dekat dengan periuk kek..
  5. Apabila masak, biarkan sejuk dan terbalikkan atas redai, jika suka tuang ganache. Sedia dinikmati, sedap lembut sangat..
  6. Coklat ganache - optional (lagi merecik ada ni).
